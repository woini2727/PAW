6) ¿Qué es un motor de persistencia ORM (Object-Relational Mapping; Mapeo objeto-relacional? ¿Qué problemática resuelve?

Un ORM te permite convertir los datos de tus objetos en un formato correcto para poder guardar la información en una base de datos (mapeo) creándose una base de datos virtual donde los datos que se encuentran en nuestra aplicación, quedan vinculados a la base de datos (persistencia). Es muy trabajoso transformar toda la información que se recibe de la base datos, en los objetos de tu aplicación y viceversa. A ésto se le denomina mapeo. Utilizando un ORM este mapeo será automático, es más, será independiente de la base de datos que estés utilizando en ese momento pudiendo cambiar de motor de base de datos según tus necesidades.
